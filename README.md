# bedwar

The `bedwar.mcddon` is mc addon base on mbler and mcx.

## Usage

### Self-host build

1. Download [Nodejs](https://nodejs.org)
2. Download pnpm

```bash
npm i -g pnpm
```

3. Download Deps

```bash
pnpm install
```

4. Build

```bash
pnpm build
```

5.

### Download from Github release

1. Download mcaddon file in Github Release
2. Open with Minecraft.
